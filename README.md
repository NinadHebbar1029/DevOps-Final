# Hotel Order Management Website

This repo is a small hotel or restaurant order management website with a React frontend, an Express backend, and MySQL storage.

## Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MySQL
- Frontend hosting: Vercel
- Backend hosting: Render
- Database: MySQL

## Local run

1. Copy `.env.example` to `.env` and fill in values.
2. Install dependencies.
3. Run the app with `npm run dev`.
